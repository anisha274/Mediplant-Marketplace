# 🌿 Mediplant Marketplace (JavaFX)

A desktop-based marketplace application developed using JavaFX that connects buyers and sellers of medicinal plants through an interactive graphical user interface.

---

## 📌 Project Overview

Mediplant Marketplace is a JavaFX-based desktop application designed to simplify the process of buying and selling medicinal plants. The system provides separate dashboards for buyers and sellers, ensuring smooth interaction and management.

---

## 🎯 Objectives

* Build a user-friendly desktop application
* Enable buying and selling of medicinal plants
* Provide role-based access (Buyer & Seller)
* Implement interactive UI using JavaFX

---

## 🛠️ Technologies Used

* Java (JDK 17 or higher)
* JavaFX
* CSS (for UI styling)
* Scene Builder (for UI design)
* IntelliJ IDEA

---

## ⚙️ Features

* User Login & Registration UI
* Buyer Dashboard (browse & purchase plants)
* Seller Dashboard (add/manage products)
* GUI built with JavaFX (FXML + CSS)
* Clean and responsive interface

## 🚀 How to Run

1. Open the project in IntelliJ IDEA
2. Make sure JavaFX SDK is configured
3. Run `Main.java`

---

## ⚠️ JavaFX Setup (Important)

Make sure VM options are set:

```
--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml
```
